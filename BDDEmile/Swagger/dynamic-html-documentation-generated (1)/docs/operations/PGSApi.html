pgsBadgesBadgeidDelete

Permet de supprimer un badge de la base de donnée et donc supprimer son association avec l'individu

URL

https://virtserver.swaggerhub.com/student-fbe/BDD/1.0.0/pgs/badges/{badgeid}

HTTP Method

DELETE

Response Type

Parameters

  • Path: badgeid String

Parameters
